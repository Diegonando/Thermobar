This folder contains files which benchmarks to existing tools for Liquid-only and Ol-Liquid equilibrium

Ca_Hygrometer.ipynb benchmarks Thermobar to the supporting Table in Gavrilenko et al. (2016). Reads from Olivine_Liquid_hygrometer.xlsx

Testing_Liquid_Ol_Thermometers.ipynb compares Thermobar to various calculations in spreadsheets from Keith Putirka's website. 
Reads from PythonInput_OlLiq_Thermometers_Putirka.xlsx. ipynb. notebook laid out so first bit is Liq-only, then ol-liq thermometers. 
This notebook also benchmarks to results from Pu et al. (2017) and (2019)
